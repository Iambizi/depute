---
title: BranchControls
description: Scoped pause, resume, quarantine, and cancel for one branch of the agent tree.
---

## Install

```bash
npx ax-depute add branch-controls
```

## Overview

RunControls are global. BranchControls are scoped to a single node — pause a troubled branch without freezing the whole swarm.
