---
title: DelegationGate
description: Approval surface for spawning a new autonomous agent.
---

## Install

```bash
npx ax-depute add delegation-gate
```

## Overview

Semantically different from ApprovalGate: approves the creation of an autonomous entity, not a single action. Shows mandate, tool constraints, estimated token cost, and lifespan.
