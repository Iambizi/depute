---
title: EscalationRouter
description: Alert feed for sub-agent failures with routing options.
---

## Install

```bash
npx ax-depute add escalation-router
```

## Overview

When a leaf node fails, the Orchestrator can retry, reassign, or escalate. EscalationRouter surfaces the failure trace and resolution options.
