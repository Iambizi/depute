## Install

```bash
npx ax-depute add orchestrator-view
```

## Overview

Tree graph of the Orchestrator and all spawned workers. Blocked nodes are highlighted. Click a node to drill into its v0 panel (PlanCard + ToolTrace + ApprovalGate).
