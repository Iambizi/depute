title: RunControls
description: Let users pause, resume, or stop a running agent.
---

## Install

```bash
npx ax-depute add run-controls
```

## Overview

Minimal execution steering — pause, resume, stop, retry. Intentionally kept simple. Extend via the actions slot when UndoStack ships.
