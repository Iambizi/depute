title: SubagentCard
description: Compact card for a single sub-agent — status, current step, confidence.
---

## Install

```bash
npx ax-depute add subagent-card
```

## Overview

Embeds inside OrchestratorView nodes or AgentRoster rows. Shows current step, ConfidenceMeter badge, and a pulsing status dot.
