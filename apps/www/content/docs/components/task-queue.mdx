title: TaskQueue
description: Backlog of pending sub-tasks with priority and reordering.

## Install

```bash
npx ax-depute add task-queue
```

## Overview

Kanban-style list of pending intents before they're delegated to workers. Supports manual injection, reordering, and per-task pause.
